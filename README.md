# Проект Movies
Бэкенд дипломной работы на курсе Web-разработчик в Я.Практикуме.

## Функционал бэкенда:
- создание баз данных пользователей и сохраненных фильмов,
- регистрация и авторизация пользователя,
- создание и удаление сохраненных фильмов.

## Стек бэкенда: 
- Node.js,
- Express,
- MongoDB,
- Ubuntu.
